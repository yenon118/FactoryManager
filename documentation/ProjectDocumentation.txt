LoginController class, RegistrationController class, and MainController class are subclasses of SceneSwitcher class.
SceneSwitcher class is an abstract class.
Transaction Interface is used in ImportationModal class, DeliveryModal class, and ExportationModal class.

Collection classes ArrayList<>, ObservableList<>, and HashMap<> are used in all the data access layer(DAL) and business logic layer(BLL).
All the database queries have excqeption handling, see all the data access layer(DAL).

Modal:
1) ImportationModal
2) DeliveryModal
3) ExportationModal
4) RegistrationModal

About were coded in MainController class.

MySQL database under Amazon Rational Database Services(RDS) is used in this project.
